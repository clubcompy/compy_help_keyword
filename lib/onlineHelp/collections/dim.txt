DIM - create a multi-dimensional array
filled with zeroes

Grammar:
 "DIM" "(" len_1 [ ... "," len_n ] ")"

Arguments:
 - len_1 - len_n: <numbers> 1 or more
   array lengths, where each len number
   indicates one dimension's size in
   the evaluated multi-dimensional array

Evaluates to:
 <array> of len_1 size, containing first
 dimension of multi-dimensional array

See also:
 %DIMS%
