GETKEY - Read a keypress from the
keyboard as a character code

Grammar:
 "GETKEY"

Evaluates to:
 <number> Kid-SCII character code
 matching the pressed keyboard key

Notes:
 - The current Comfy running task will
   pause at GETKEY until the user
   presses a keyboard key.  Other tasks
   like LISTEN tasks will continue to
   run
 - GETKEY is only allowed in the main
   Comfy task and not in LISTEN or other
   secondary tasks
 - Only keyboard keys that are
   associated with a PRINT-able Kid-SCII
   character's code will flow out of
   GETKEY
